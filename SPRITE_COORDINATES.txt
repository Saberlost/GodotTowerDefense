## How to Use This Configuration

### For Enemy Sprites (sprits_monsters.jpg):

1. Create an AtlasTexture resource in Godot:
   - Right-click in FileSystem panel
   - Create New Resource -> AtlasTexture
   - Set the atlas property to sprits_monsters.jpg
   - Set the region to the coordinates listed above

2. Use the AtlasTexture in enemy scene:
   - Open the enemy .tscn file
   - Replace Polygon2D with Sprite2D
   - Assign the AtlasTexture to the texture property

### For Tilemap Sprites (tilemap.jpg):

1. Create AtlasTexture resources for each tile type:
   - Create separate AtlasTexture for grass, path, etc.
   - Set regions according to the coordinates above

2. Update main.gd to use sprites:
   - Preload the AtlasTexture resources
   - Modify draw_tile() to create Sprite2D instead of ColorRect
   - Pass appropriate texture based on tile type

3. Alternative (Advanced): Create TileSet resource:
   - Create a new TileSet resource
   - Add tilemap.jpg as the source
   - Define each tile with its region
   - Use TileMap node in main scene
